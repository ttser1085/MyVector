# MyVector

Реализация стандартного контейнера, аналогичного `std::vector`, и своего *Random Access* итератора. Контейнер поддерживает использование произвольного аллокатора памяти.

### Сборка тестов:
```bash
mkdir build
cd build

cmake ..
make
```
